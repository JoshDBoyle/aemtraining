Queue Status Legend
Color Meaning
Agent is enabled, active (not in standby), and has no queue
Agent is enabled, active, and the queue is unblocked and actively replicating content
Agent is enabled, active, and actively replicating content but its queue is blocked
Agent is in standby mode. Replication requests will continue to queue up as normal but nothing will be replicated until the agent is no longer in standby and the remaining duration of its retry period has elapsed
Agent is disabled. Replication requests will be ignored altogether by this Agent.
